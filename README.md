### Download album from youtube, split into tracks and tag them

# prerequisites

```
pip install pillow
pip install pydub
pip install pymad
pip install youtube-dl
```

# On Debian

```
apt install ffmpeg
apt install id3v2
```

# On Arch
```
pacman -S ffmpeg
pacman -S id3v2
```

# Usage

```
python splitalbum.py
```
